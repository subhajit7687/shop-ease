# E-Commerce app

This app is to practice basic flutter skills. At the same time explore complex UI design patterns,
learn how to integrate google firebase and explore more advanced concepts like go_router, bloc state
management, dependency inject any many more.

> Following the tutorial
> from [Coding With T](https://www.youtube.com/playlist?list=PL5jb9EteFAOAusKTSuJ5eRl1BapQmMDT6)

While following the videos from above-mentioned channel, not everything is copied as it is.
While the UI and logics are heavily inspired from the tutorials, but many implementation logics are
my own.
